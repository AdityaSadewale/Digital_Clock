Digital Clock Project

A simple and stylish Digital Clock created using HTML, CSS, and JavaScript.
This project shows the current time in hours, minutes, and seconds and updates automatically in real time ⌚✨

🌟 Features

🕒 Displays real-time digital clock

🎨 Clean and user-friendly design

⚡ Fast and lightweight

💻 Beginner-friendly project

🌙 Easy to customize

🛠️ Technologies Used

🧱 HTML

🎨 CSS

⚙️ JavaScript

📂 Project Structure

📄 index.html – Main HTML file

🎨 style.css – Styling and layout

⚙️ scripts.js – Clock logic and functionality

🎯 Project Purpose

This project helps beginners understand:

⏱️ Time and date functions in JavaScript

🧩 DOM manipulation

🎨 Basic web design concepts

📸 Preview

⏰ A live digital clock that updates every second with a clean UI
